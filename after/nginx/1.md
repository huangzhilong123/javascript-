# 命令

```
nginx -t ## 查看配置文件目录
```

```
start nginx      // 开启nginx
nginx -t         // 检查nginx配置文件
nginx -V         // 查看版本信息
nginx -s reload  // 重新加载配置
nginx -s quit    // 关闭nginx
nginx -s stop    // 停止nginx
service nginx stop      // 停止
service nginx start     // 启动
service nginx restart   // 重启
```
